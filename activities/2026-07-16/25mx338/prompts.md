# Prompts Used

## Prompt 1

Explain the complete GitHub workflow for contributing to an open-source repository, including fork, clone, commit, push, and pull request.

## Prompt 2

Explain the purpose of a Pull Request and how it helps teams collaborate on software projects.

## Prompt 3

Provide the recommended folder structure and steps to complete a GitHub contribution task without modifying files outside the assigned directory.

## Prompt 4

Explain common Git errors during push operations and how to troubleshoot them.
